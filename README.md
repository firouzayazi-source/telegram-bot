# ربات + پنل وب استوک لند (متصل به ووکامرس)

ربات تلگرام فروشگاه موبایل + پنل مدیریت وب، که محصولاتش را زنده از سایت **ووکامرس** (stland.ir) می‌خواند.

محصولات فقط در **وردپرس** مدیریت می‌شوند. ربات و پنل وب آن‌ها را نمایش می‌دهند (فقط‌خواندنی).

---

## ساختار فایل‌ها

```
.
├── app.py              ← راه‌انداز اصلی (بات + وب با هم)
├── bot.py              ← ربات تلگرام
├── requirements.txt
├── Procfile
├── railway.json
└── webpanel/
    ├── web.py          ← سرور پنل وب
    ├── templates.py    ← قالب‌های HTML
    ├── woo.py          ← اتصال به ووکامرس
    └── uploads/        ← عکس بنرها
```

---

## مرحله ۱: ساخت کلید API ووکامرس

۱. وارد پنل وردپرس شو → **WooCommerce → Settings → Advanced → REST API**
۲. **Add key** را بزن
۳. توضیح: telegram-bot | کاربر: ادمین | دسترسی: **Read**
۴. **Generate API key** → دو کلید می‌گیری:
   - Consumer Key → مقدار WOO_KEY
   - Consumer Secret → مقدار WOO_SECRET

> کلیدها فقط یک بار نمایش داده می‌شوند. همان لحظه کپی کن. سایت باید روی HTTPS باشد.

---

## مرحله ۲: متغیرهای محیطی (Railway → Variables)

| نام | توضیح | مثال |
|-----|-------|------|
| BOT_TOKEN | توکن ربات از BotFather | 123456:ABC... |
| ADMIN_ID | آیدی عددی ادمین | 123456789 |
| PANEL_USER | نام کاربری پنل وب | admin |
| PANEL_PASS | رمز عبور پنل وب | یک رمز قوی |
| WOO_URL | آدرس سایت | https://stland.ir |
| WOO_KEY | Consumer Key | ck_xxxxx |
| WOO_SECRET | Consumer Secret | cs_xxxxx |
| WOO_CACHE_TTL | مدت کش (ثانیه) — اختیاری | 3600 |
| WOO_HIDE_OOS | مخفی‌کردن ناموجودها — اختیاری | 1 |
| PANEL_SECRET | کلید نشست — اختیاری | یک رشته تصادفی |

---

## مرحله ۳: راه‌اندازی روی Railway

1. کل پوشه را در GitHub بگذار.
2. در Railway پروژه از روی ریپو بساز.
3. متغیرهای بالا را در Variables وارد کن.
4. در Settings → Networking یک دامنه عمومی بساز.
5. دامنه را باز کن → صفحه ورود پنل ظاهر می‌شود.

ربات و پنل با یک سرویس اجرا می‌شوند.

---

## نحوه کار با محصولات

- افزودن/ویرایش/حذف محصول: فقط در پنل وردپرس سایت.
- ربات و پنل وب: محصولات را زنده نمایش می‌دهند.
- کش یک‌ساعته: تغییرات سایت تا یک ساعت بعد در ربات دیده می‌شوند. برای دیدن فوری، دکمه بروزرسانی را بزن.
- محصولات ناموجود به‌طور پیش‌فرض نمایش داده نمی‌شوند.

نگاشت دسته‌بندی:
```
دسته اصلی ووکامرس  →  دسته اصلی ربات
زیردسته ووکامرس    →  زیردسته ربات
محصولات            →  از ووکامرس
```

## اجرای محلی (تست)

```bash
export WOO_URL="https://stland.ir" WOO_KEY="ck_..." WOO_SECRET="cs_..."
python app.py
```

مرورگر: http://localhost:8080
